
# Stellia Test Practice

Proyecto hecho sobre las tecnologias:

- React JS
- Material Ui ( MUI ) 
- Admin de estado ( Redux Toolkit)
- Css Puro
- Firabase y FireStore para el manejo de llamadas 
- Bundler ( Vite )
